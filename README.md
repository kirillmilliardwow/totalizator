# 🎲 Магический Рандомайзер

Красивое приложение-рандомайзер с анимациями для Telegram Mini Apps и веба.

![Preview](preview.gif)

## ✨ Особенности

- 🎯 Рандомайзер "Да/Нет/Наверное" с красивой анимацией колеса
- 🌟 8 вариантов ответов с уникальными сообщениями
- 💫 Пожелания прекрасного дня
- 🎨 Анимации частиц и конфетти
- 🌓 Тёмная и светлая темы
- 📱 Адаптивный дизайн
- 🔊 Звуковые эффекты
- 💾 История вопросов
- 📤 Поделиться результатами
